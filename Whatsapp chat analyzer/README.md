How to use this app?

1.Go to your WhatsApp group chat or private chat. On top right corner, click on the three dots to open the menu. Select 'More' and then 'Export Chat'. Choose 'Without Media' option to export the chat.
2.A zip file will be downloaded. Extract the zip file to get the .txt chat file.
3.Upload the .txt file in this app.
4.Select the user from the dropdown to analyze individual chat or select 'Overall' to analyze the whole chat.
5.Click on 'Show Analysis' button to see the statistics and visualizations.
6.Enjoy analyzing your WhatsApp chats.
